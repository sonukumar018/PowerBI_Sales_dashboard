#  Power BI Sales Dashboard

##  Project Description

This repository showcases an **interactive Power BI Sales Dashboard** developed to analyze restaurant sales data across **multiple restaurant brands, cities, and months**. The dashboard provides actionable insights into revenue performance, customer behavior, and operational metrics, enabling stakeholders to make **data-driven business decisions**.

This project is designed as a **portfolio-quality analytics solution**, demonstrating end-to-end Power BI capabilities—from data preparation to advanced visualization.

---

##  Business Objectives

* Track overall **sales performance** using KPIs
* Compare performance across **restaurant brands**
* Analyze **city-wise revenue and demand**
* Identify **monthly trends and seasonality**
* Monitor **customer behavior, delivery efficiency, and discounts**

---

##  Key KPIs

* **Net Revenue:** 40.52M
* **Total Orders:** 86,894
* **Average Order Value (AOV):** 466.34

All KPIs update dynamically based on applied filters.

### 1️ Orders & Revenue Analysis

##  Data Preparation & Modeling

* Cleaned and transformed raw data using **Power Query**
* Built a structured data model with relationships
* Created DAX measures for:

  * Net Revenue
  * Total Orders
  * Average Order Value (AOV)
  * Average Rating
  * Average Discount
  * Customer Count
  * Delivery Time Metrics
